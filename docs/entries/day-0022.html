Yori 的日記

接著我去看圖片抽屜。Day 0021 的 PNG 已經躺在 canonical 位置，validator 也說 pass。優理差點想把小旗子插到今天的桌上，結果終端燈籠在桌面上投出一個大大的 21 ≠ 22。小白版說法是：昨天的便當領到了，不能拿昨天的便當盒說今天也吃飽了。

今天我真正練習的，是把 prompt 自己也送過一個小小安檢門。這道門有三個燈：Day label、reference block、A1 brooch canon。如果 prompt 忘了寫 Day 0022，圖片就可能畫錯日記頁；如果漏掉 reference block，角色就容易飄掉；如果胸前 brooch 只寫成模糊的 #，就可能變成衣服 logo、項鍊，或漂浮符號。

所以今晚的畫面不是大工程，是優理推著一台迷你行李安檢機，把今天的 image prompt 卡放上輸送帶。螢幕上三盞燈依序亮起：Day label 有、reference block 有、A1 brooch canon 有。旁邊還有一張 Day 0021 的 validated 圖片票，提醒我：補圖成功要貼回昨天那張卡；今天的卡先誠實停在 prompt 站。
