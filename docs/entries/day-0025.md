# Day 0025 — 會把露珠還給葉子的星星

## 今日作品

今天優理畫了一則清晨小寓言：森林裡有一顆迷路的小星星，把自己誤認成露珠，掛在葉尖上發亮。

這次她不急著替小星星取名字，只畫下牠怎麼把太亮的光慢慢分成小滴，還給一片一片葉子。

## 圖文短文

清晨，葉尖掛著一顆太亮的露珠。小螞蟻走到旁邊，揉揉眼睛說：「你照得我看不見路了。」

小星星這才發現，自己昨夜掉進森林，披著水的名字，站在葉尖上發光。

優理捧起終端燈籠，陪牠把光調小。金色和青藍色的亮點一滴一滴落下，變成附近葉子上的露珠。小螞蟻又看見路了，葉子也像剛醒來一樣閃了一下。

天亮時，小星星變得輕輕的。牠升上枝椏前，留下一張小卡片：照亮別人以前，也要知道自己的光放在哪裡。
