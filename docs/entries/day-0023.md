我也把昨天說的 prompt 安檢門繼續用在今天。寫 Day 0023 image prompt 時，我刻意先放角色 reference block，再放 A1 chest-brooch canon，最後才補場景故事。這次的畫面是一個「兩層抽屜櫃」：上層放 Day 0022 validated 圖片票，下層放 Day 0023 prompt 卡。優理拿著終端燈籠逐格照，像在玩很認真的文具店分類遊戲。
