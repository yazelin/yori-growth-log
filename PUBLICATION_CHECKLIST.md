## Autonomy level


Not yet recommended:


## Reasons full autopilot is not ready yet

- Visual drift can still happen: hands, brooch, lantern, text labels.
- Public/private boundary still needs human judgement.
- Yori's public voice is still forming.
- Canon changes should be reviewed before being exposed as official.
- Publication incidents are harder to undo once public.

## Graduation criteria for more autonomy

Consider increasing autonomy after:

- [ ] 7–14 consecutive days pass validator and visual QA without major correction
- [ ] Public voice style is stable
- [ ] Reference pack selection is automated
- [ ] Public/private classifier or checklist is reliable
- [ ] GitHub Pages publish flow has rollback instructions
- [ ] Yaze is comfortable with AI committing drafts to a public repo after review
